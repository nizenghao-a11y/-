# 练对啦产品需求文档 V1

## 一、底部 Tab 架构

### 1. 练习

- 8 肌群分类动作库
- 教练 / 专业用户上传标准动作
- 动作赞赏
- 视频讲解
- 篮球投篮专项动作分析与教学

### 2. 练对

- 付费发布视频点评订单
- 达人 / 教练 / 专业用户接单答疑
- 平台托管佣金结算
- 篮球投篮动作点评订单

### 3. 论坛

- 用户发帖晒训练
- 动作互评投票：`练对啦 / 有问题`
- 健身、体态、康复、功能训练、运动营养、篮球技术等内容交流
- 论坛仅保留技术交流属性，不保留社交私聊属性

### 4. 我的

- 账单流水
- 我的订单
- 教练动作管理
- 余额提现
- 资料审核
- 客服记录

## 二、全用户通用基础规则

### 1. 登录

- 微信一键快捷授权登录
- 自动生成独立个人主页

### 2. 私信管控

- 用户与点评人无法互相发送私信
- 平台剔除私聊社交属性
- 禁止通过平台进行与训练无关的导流、暧昧社交、灰色交易

### 3. 论坛定位

- 论坛只做技术交流，不做泛社交社区
- 允许内容：
  - 动作分析
  - 训练记录
  - 体态讨论
  - 康复经验
  - 篮球技术
  - 功能训练
  - 饮食营养交流
  - 瑜伽 / 普拉提训练经验
- 禁止内容：
  - 留联系方式
  - 约线下
  - 与训练无关的情感社交
  - 擦边、低俗、导流内容

## 三、身份体系

身份体系拆分为两层：`基础等级` + `专业身份`

### 1. 基础等级

- 小白
- 老手
- 大神

### 2. 基础等级说明

#### 小白

- 可浏览动作库
- 可赞赏动作
- 可发布付费提问
- 可浏览论坛技术内容
- 可发技术交流帖
- 可收藏内容
- 不可接单
- 不可上传标准动作

#### 老手

- 拥有小白全部权限
- 可申请升级接单权限
- 审核通过后可在练对接单

#### 大神

- 拥有老手全部权限
- 可申请上传标准动作
- 可申请专业身份认证
- 展示权重更高

### 3. 基础等级升级规则

- 默认注册为小白
- 用户可主动提交升级申请
- 升级路径：
  - 小白 -> 老手
  - 老手 -> 大神
- 升级需由后台人工审核通过后生效

## 五、篮球专项接单体系

篮球不作为独立资质身份，不设置强制的 `篮球教练证书审核`。

平台只做 `投篮动作分析功能`，采用 `内容沉淀 + 素材佐证 + 历史履约口碑` 控制接单权限。

### 1. 三类接单权限划分

#### 普通篮球爱好者（小白）

开通条件：

- 仅勾选 `篮球（投篮专项）` 强项标签
- 无任何历史数据要求

限制规则：

- 禁止主动接单投篮订单
- 可自己发布投篮提问
- 可浏览别人点评内容
- 不可通过投篮点评赚佣金

#### 资深篮球爱好者（老手）

开通条件满足 2 选 1：

- 个人主页上传篮球训练实拍或打球视频，后台只核验素材真实，不核验证书
- 在论坛累计发布不少于 15 条篮球相关优质帖子，且点赞数据达标

限制规则：

- 可接单低价小额投篮订单
- 限定单价为 `3 元 / 5 元`
- 不可接用户自定义金额高价悬赏单
- 平台抽佣 20%
- 单日接单上限 5 单

#### 篮球专项大神

开通条件满足任意两项：

- 上传赛场实拍或业余赛事参赛证明，如校园赛、民间联赛等
- 篮球类历史点评订单好评率大于等于 85%
- 累计完成不少于 30 单投篮点评订单

限制规则：

- 可接用户自定义金额高价悬赏单
- 无低价限制
- 单日接单不设上限

### 2. 用户发单筛选

用户发布投篮分析订单时，可勾选：

- `仅大神接单`

规则如下：

- 勾选后：订单自动屏蔽资深篮球爱好者，仅篮球专项大神可接
- 不勾选：资深篮球爱好者与篮球专项大神都可接

### 3. 差评熔断机制

同一接单者满足以下任一条件时，冻结投篮接单权限 7 天：

- 连续 3 单差评
- 用户退款申诉成功不少于 2 单

恢复条件：

- 重新补充篮球相关实拍素材
- 后台复审通过后恢复接单

### 4. 内容交付要求

投篮点评必须包含实质性内容，至少满足以下之一：

- 实拍纠正视频
- 分段动作拆解图文
- 明确的出手、发力、稳定性建议

违规处理：

- 用户可申请全额退款
- 平台扣减账号评分
- 多次违规永久关闭篮球接单权限

### 5. 篮球板块联动

#### 练习

- 新增 `篮球投篮` 分类
- 收录标准投篮 GIF 教学
- 支持投篮动作讲解
- 认证大神可上传投篮教学动作并接受赞赏
- 赞赏沿用平台 10% 抽成规则

#### 论坛

- 新增篮球分区
- 篮球内容可作为爱好者积累帖子数量的数据来源
- 篮球内容可作为解锁篮球接单权限的参考数据

#### 后台审核

- 篮球相关素材仅核验是否盗用网图、是否虚假、是否违规
- 不核验篮球教练员证书

## 六、个人主页规则

### 1. 主页定位

- 个人主页定位为专业能力名片
- 不做社交展示页
- 不做背景图装饰玩法

### 2. 主页展示字段

#### 基础信息

- 头像
- 昵称
- 性别（可选）
- 城市（可选）
- 训练年限
- 个人简介（简短说明）

#### 身份信息

- 基础等级：小白 / 老手 / 大神
- 专业身份标签
- 专项方向标签
- 是否可接单
- 是否可上传标准动作

#### 能力信息

- 强项身体部位标签
- 强项专项标签
- 历史接单量
- 好评率
- 标准动作上传数量
- 投篮专项接单状态

#### 信誉信息

- 主页点赞数
- 主页浏览量
- 累计帮助人数
- 审核状态

#### 审核资料展示

- 身材实拍
- 获奖证明

- 待审核

- 不支持主页背景图
- 不强化粉丝社交体系
- 不突出“关注 / 粉丝”社交关系

## 八、后台审核模型

后台审核统一采用 `审核中心` 模型，不同审核对象共用同一套状态机。

### 1. 审核对象类型

- 基础等级升级申请
- 专业身份认证申请
- 身材实拍
- 获奖证明
- 训练履历
- 头像
- 标准动作上传
- 篮球相关素材
- 退款申诉

### 2. 通用审核状态

- 待审核
- 审核通过
- 审核驳回
- 冻结中
- 已关闭

### 3. 审核记录字段

- 审核记录 ID
- 用户 ID
- 审核对象类型
- 审核对象 ID
- 提交时间
- 当前状态
- 驳回原因
- 审核人
- 审核时间
- 备注

### 4. 身份升级审核模型

#### 小白 -> 老手

审核依据：

- 训练时长说明
- 实拍训练内容
- 平台活跃记录

#### 老手 -> 大神

审核依据：

- 历史优质内容
- 历史订单口碑
- 训练履历
- 相关证明材料

### 5. 专业身份审核模型

认证对象：

- 认证教练
- 康复师
- 体态矫正师
- 运动营养师
- 瑜伽师
- 普拉提教练
- 体能教练
- 体育老师

审核依据：

- 专业证书
- 教学履历
- 案例资料
- 平台历史表现

### 6. 动作上传审核模型

审核项：

- 内容是否违规
- 动作是否真实
- 是否盗用他人素材
- 标题与内容是否一致
- 是否具备教学价值

### 7. 篮球专项审核模型

篮球专项不做证书审核，采用低门槛素材审核：

- 是否为真实训练素材
- 是否盗用网图 / 网视频
- 是否与投篮专项相关
- 是否具备基础分析价值

### 8. 退款审核模型

退款审核需记录：

- 订单编号
- 申请人
- 退款理由
- 证据材料
- 接单方交付内容
- 审核结果
- 是否全额退款 / 部分退款 / 驳回

## 九、接单与上传权限

### 1. 可接单用户

以下身份在审核通过后可接单：

- 老手
- 大神
- 认证教练
- 康复师
- 体态矫正师
- 运动营养师
- 瑜伽师
- 普拉提教练
- 体能教练
- 体育老师

### 2. 可上传标准动作用户

以下身份在审核通过后可上传标准动作：

- 认证教练
- 瑜伽师
- 普拉提教练
- 体能教练
- 体育老师
- 体态矫正师

康复师和运动营养师默认不强制开放标准动作上传，可按后续运营策略决定：

- 康复师：可优先开放康复训练演示动作
- 运动营养师：默认不上传标准动作视频，以科普内容为主

## 十、后续筛选体系建议

在练对和练习中，可按以下维度筛选用户：

- 基础等级
- 专业身份
- 专项方向
- 强项部位
- 历史接单量
- 好评率
- 是否已认证

## 十一、当前版本结论

当前平台身份体系固定为：

- 基础等级：`小白 / 老手 / 大神`
- 专业身份：`健身爱好者 / 认证教练 / 康复师 / 体态矫正师 / 运动营养师 / 瑜伽师 / 普拉提教练 / 体能教练 / 体育老师`
- 专项方向支持继续扩展：`功能训练 / 康复 / 瑜伽 / 体能 / 青少年训练 / 投篮动作分析`
- 篮球作为专项动作分析场景独立存在，先做功能，不做独立资质身份认证

后续所有页面、权限系统、订单系统与后台审核逻辑，默认基于这套规则推进。
